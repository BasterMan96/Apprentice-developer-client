# Байтик — Frontend

Клиентская часть платформы для изучения программирования.

## Стек

- React 19 + TypeScript
- Vite 6
- Tailwind CSS 3 (mobile-first)
- Zustand 5 (state management)
- React Router 7
- CodeMirror 6 (редактор кода)
- Framer Motion (анимации)
- Axios (HTTP клиент)

## Запуск

Смотри инструкцию в корневом README проекта.

## Локальная разработка

```bash
npm install
npm run dev
```

Требуется запущенный backend на http://localhost:8080 (Vite проксирует /api).
